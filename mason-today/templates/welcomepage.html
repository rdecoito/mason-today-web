MasonToday API

We have two APIs available, pulled from two different sources of GMU events:

  • Get Connected: masontoday.gmu.io/api/getconnected
  • 25 Live: masontoday.gmu.io/api/25live

Welcome to the API for MasonToday, courtesy of Student Run Computing and Technology! If you'd like, you can find the source code at the GMU Gitlab, https://git.gmu.edu/srct/mason-today-web
